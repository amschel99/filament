This is a live demo of Filament — “accept stablecoin payments over Fiber in one API call.” The amount is denominated in fUSD, a UDT stablecoin minted on the devnet; the roaster’s node issues a real fUSD invoice; a buyer node pays it, routed through the hub; and the invoice flips to PAID from observed on-chain state. Everything here is running against a real CKB devnet — the same code path swaps fUSD for RUSD on mainnet.
